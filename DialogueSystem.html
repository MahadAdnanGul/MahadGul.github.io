Node Based
Dialogue System

A node based visual dialogue system to simplify complex dialogue trees

Summary:

Part of a top-down pixel art RPG game i'm currently working on. While planning for the different systems and mechanics I felt that the biggest hurdle in a narrative driven game would be handling the dialogue progression; handling dialogue choices, consequences and events triggering from dialogues that could potentially affect something that may not have already happened in a game. I saw some expensive assets on the Unity asset store and thought that it would a good idea to make something like that on my own. I have made use of Unity graph view to create a node based editor and a custom parser to allow for creating of custom dialogue assets that can be fed directly to an npc. Caters to choices, events, saving/loading of data, NPC status and relationships all in an easy to use, designer friendly package. WIP.
